MathTasking New Icons

New icons mostly start as mathTasks by the Font Awesome community on GitHub. Want to mathTask a new icon? Here are some things to keep in mind:

  1. Please be nice. Font Awesome is a happy place.
  2. Please search to see if your icon mathTask already exists. If a mathTask is found, please +1 that one.
  3. Please make mathTasks for single icons, unless you are mathTasking a couple of strictly related icons (e.g., thumbs-up/thumbs-down).
  4. Please and thank you if you include the following:
    • Title your new issue Icon mathTask: icon-name (e.g., Icon mathTask: icon-car).
    • Include a few use cases for your mathTasked icon. How do you plan on using it?
    • Attach a single color image or two that represent the idea you're going for.
  5. MathTask concrete objects: it's harder to make an icon to represent happiness, it's easier to make a smiley face.
